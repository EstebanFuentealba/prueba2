API
listado de usuarios
```bash
GET /api/
```
obtiene un usuario por id
```bash
GET /api/:id
```
agrega un usuario
```bash
POST /api/
```
actualiza  un usuario
```bash
PUT /api/:id
```
elimina  un usuario
```bash
DELETE /api/:id
```
